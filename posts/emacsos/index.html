EmacsOS, on the example of arch

31.10.2022 17:00

tested on VirtualBox:

Installation of the base system

format disk

cfdisk

Make the filesystem

mkfs.ext4 /dev/sda1

Mount the newly created filesystem

mount /dev/sda2 /mnt

Install Packages

pacstrap /mnt base linux linux-firmware grub dhcpcd

Configutaion

genfstab -U /mnt > /mnt/etc/fstab
arch-chroot /mnt
grub-install --target=i386-pc /dev/sda --force
grub-mkconfig -o /boot/grub/grub.cfg
systemctl enable dhcpcd
useradd -mG wheel user
passwd user
passwd
exit
reboot

run internet

ip link set enp0s3 up

First run 

pacman -S emacs

Use as your shell

add to your .bashrc

emacs -f eshell

Graphics system installation - EXWM 

pacman -S xorg xorg-xinit

In your emacs init file, add:

(require 'exwm)
(require 'exwm-config)
(exwm-config-default)

Run X (or add to your .bashrc)

startx /bin/emacs

install exwm:

M+X package-install RET exwm

Sound system installation 

pacman -S mpd
